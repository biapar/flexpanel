FlexPanel

A scrolling panel navigation for mobile and desktop

Tweet Follow @KaptonKaos

FlexPanel is a jQuery/CSS3 plugin for creating a scrollable, sliding panel navigation system.

Download

Features

Swipe Controls

With the help of hammer.js you can swipe to interact with the Flexpanel menu.

CSS3 Animations

There are no jQuery animations here. All animations are controlled by CSS3 properties.

Fully Responsive

From mobile applications to desktop websites, FlexPanel can handle your navigation.

Usage

Basic Init Method

FlexPanel's initiate method in it's most simplest form.

$('.flexpanel').flexpanel();

Advanced Method

Passing configuration paramaters to FlexPanel. More info.

$('.flexpanel').flexpanel({
    wrapper: '#wrapper',
    button: '.flex-btn',
	maxWidth: 1200,    
});

Public Functions

Open Try Me

//Open FlexPanel from any element on the screen.
$.fn.flexpanel.open();

Close Try Me

//Close FlexPanel from any element on the screen.
$.fn.flexpanel.close();

Toggle Try Me

//Toggle FlexPanel Open/Close.
$.fn.flexpanel.toggle();

Configuration

FlexPanel accepts the following configuration parameters...

{
    direction: 'right', // Panel slides in from 'left', 'right' or 'top'.
	wrapper: '#wrapper', // Define the main content wrapper.
	button: '.flex-btn', // Define the menu button.
	maxWidth: null, // Define a minimum screen width, this will help trigger FlexPanel functions.	
	//panelWidth: 80, // Coming soon - Percent width of the panel, default is 80%.
	speed: 500, // Define transition for CSS3 fallback.
}
